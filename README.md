# Edu-IA-IFPA
Prototipo de uma Plataforma de ensino para disciplinas técnicas baseada em API de Inteligência Artificial, voltada para a consulta de ementas e apoio pedagógico aos alunos do curso Técnico em Informática do IFPA - Campus Óbidos. Desenvolvido como Projeto Integrador.

## Sobre o Projeto
Este projeto nasce da necessidade de facilitar o acesso e a compreensão das ementas curriculares através de uma interface de chat inteligente. Ele integra conceitos de Engenharia de Software, Bancos de Dados e IA.

## Tecnologias Utilizadas
  Banco de Dados: MySQL (Modelagem Relacional)
  Modelagem: draw.io,dbdiagram.io e MySQL Workbench
  Banco de Dados: MySQL
  Design: Figma
  Front-end: HTML5, CSS3 e JavaScript (em desenvolvimento)
  Back-end: Python, Flask e PHP
  IA: Google Gemini API (planejado)
