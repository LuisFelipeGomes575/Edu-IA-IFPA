# Edu-IA-IFPA
Prototipo de uma Plataforma de ensino para disciplinas técnicas baseada em API de Inteligência Artificial, voltada para a consulta de ementas e apoio pedagógico aos alunos do curso Técnico em Informática do IFPA - Campus Óbidos. Desenvolvido como Projeto Integrador com foco em Engenharia de Software e Bancos de Dados Relacionais.
